Just Show Me the Code!

Show me yours and I'll show you mine? Ok, nevermind. I'll just show you mine.

Sdunn.net is a responsive website fully designed and developed by Steven Dunn. The front-end was built with HTML5, CSS3, JavaScript, LESS, and jQuery, and the source code is managed by a Git repository using Terminal and SourceTree. The back-end involves connecting to the Web Server via SSH and cloning instances of the repo to various sub-domains of the site (e.g. dev.sdunn.net for the development server).

The site achieves Google PageSpeed Scores of 95% and 89% (on desktop and mobile respectively) and loads faster than 98% of all tested websites according to Pingdom.com! For comparison, try testing your own website at Googleâ€™s PageSpeed Page and Pingdom (make sure to use the Dallas, Texas server).

This performance is achieved with the help of Grunt, which is used to automate the build process, optimize images, reduce HTTP requests, and minify all HTML, CSS, and JavaScript files.
