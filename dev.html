Source Code

Sdunn.net is a work in progress and is being designed and developed 100% by yours truly. The front-end was built with HTML5, CSS3, JavaScript, LESS, jQuery, jQueryMobile, and Grunt. The source code is managed using Terminal and SourceTree, and the “back-end” involves cloning instances of a Git repository to various sub-domains via SSH.

The site achieves Google PageSpeed Scores of 95% and 89% (on desktop and mobile respectively) and loads faster than 98% of all tested websites according to Pingdom.com! For comparison, try testing your own website at Google’s PageSpeed Page and Pingdom (make sure to use the Dallas, Texas server).

This performance is achieved by using Grunt to optimize images, reduce HTTP requests, and minify all HTML, CSS, and JavaScript files.
